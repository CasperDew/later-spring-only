## Учебный проект Later. 
В ходе разработки используется `Spring Framework`.</br>
Дополнительно используются:

- Tomcat - для запуска приложения
- Jackson - для работы с форматом JSON
- Logback - для ведения логирования работы приложения

Для работы с базой данных:
- Spring data jpa 
- Hibernate - реализация интерфейса JPA
- QueryDSL - для конструкции запросов
